# Cardiac-Multi-view-EchO-Dataset-CAMEO
![image](https://github.com/rayleikai/Cardiac-Multi-view-EchO-Dataset-CAMEO/blob/main/CAMEO_Banner.png)
The source code and scripts for the dataset visualization and training/testing split are publicly available

To reproduce the view classification experiments using this dataset, please strictly follow the dataset protocol.
This ensures that all models are evaluated under the same benchmark settings and that the results remain comparable across studies.

If you use this dataset in your research or publications, please cite the following paper:

S. -Y. Lee, K. -Z. Lei, J. -Y. Chen and C. -R. Huang, "Descriptor: Cardiac Multi-View EchO Dataset (CAMEO)," in IEEE Data Descriptions, vol. 3, pp. 47-53, 2026, and the dataset is available at IEEE Dataport, August 15, 2025, doi:10.21227/kvaz-eh57.
